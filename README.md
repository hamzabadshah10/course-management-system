## 👩‍💻 About This Repository

This project provides a straightforward, accessible web interface for administrators and students to seamlessly add, view, and manage university courses. By migrating manual course catalogs into a digitized, relational database structure, it ensures higher efficiency and accurate record keeping. 

---

## 🚀 Key Features

* **Interactive Course Addition:** Intuitive web forms to input essential course details including Course Code, Name, Instructor, and Credits.
* **Dynamic Catalog Dashboard:** A responsive, real-time dashboard listing all available courses pulled securely from the MySQL database.
* **Efficient Data Handling:** Built upon a standard Client-Server architecture utilizing standard PHP data objects (PDO/MySQLi) for secure database interactions.
* **Clean & Responsive UI:** Crafted with HTML, CSS, and JS to ensure the interface looks great on both desktops and mobile devices.

## 🚀 Quick Start

1. **Clone the Repository:**
   Place the project folder into your local development server's web directory (e.g., `C:\xampp\htdocs\` or `C:\wamp64\www\`).
   ```bash
   git clone https://github.com/hamzabadshah10/course-management-system.git
   ```

2. **Database Configuration:**
   * Open your MySQL client or **phpMyAdmin** (`http://localhost/phpmyadmin`).
   * Create a new database (e.g., `course_registration`).
   * Run the provided `database.sql` script to generate the required schema.
   * Verify that your credentials in `db.php` match your local setup.

3. **Launch the Application:**
   * Start your Apache and MySQL servers.
   * Access the portal at `http://localhost/course-management-system/index.php`.
